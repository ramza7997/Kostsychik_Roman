Блок навигации
  • Ширина 15ем.
  • Должен присутствовать цвет фона.
  • Распологается слева
  • Блок навигации наполните
    контентом достаточным,
    чтобы по высоте он начинал
    захватывать второю статью,
    как показано в данном примере.
Статья 1

Размер блока:
Занимают всю ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным 2ем,
При помощи margin-bottom
Статья 2

Размер блока:
Занимают всю ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным 2ем,
При помощи margin-bottom. Картинка вставлена при помощи тега, плавание
организованно по правому краю внутри данной статья для этой картинки.
Нижний блок, подвал. Занимает всю ширину. Высотаопределяется помещенным внутрь контентом.
